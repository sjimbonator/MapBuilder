Overview

Mapcraft makes it easy for users to select and edit an image of their choice to share it with others by sending the url that is generated when editing an image. It uses the OpenLayers API to convert the chosen image to be able to use the mapping functions of the API for editing.



Quick start

To use MapCraft, you will have to be logged in. Create an account now by clicking the link above.

Once logged in, you can select and upload an image to start editing!

Viewing someone's map is as easy as pasting the map url created by the original editor.
